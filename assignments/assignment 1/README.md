
# Build and Serve SPA Application


# Prerequisites

 The following tools, frameworks, and modules are required for this tutorial:

 - [Node.js](https://nodejs.org/en/) (recommended version)
 - [Angular 7 CLI](https://cli.angular.io/)
 - [Angular 7](https://angular.io/)

We assume that you have installed Node.js. Now, we need to check the Node.js and NPM versions. 
    
    node -v
    vx.x.x
    npm -v
    x.x.x
Now, you can go to the main steps.

# Challange

You have to create an angular application with three pages, 
 - Home
 - About us
 - Contact us

 # Expected Result to submit 
  - You have to follow angular best practices
  - Solution should involve angular routing
  - Should prevent flicker when Angular route change
